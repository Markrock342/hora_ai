## สรุปสั้นๆ

**ชั้นดาราศาสตร์:** `astronomy-engine` + อายนางศา **ลาหิรี** (`utils/formulas/lahiri.ts`)  
**ชั้นไทย (โครง):** ราหู/เกตุ = ลูกหลุม, ทักษา 9 ช่อง, เรือน Whole Sign — **ปรับจูนกับ myhora** ผ่าน `docs/test-cases.json`  
**แหล่งอ้างอิงสูตรเต็ม:** ซอร์ส [myhora.com/astrology/thai.aspx](https://myhora.com/astrology/thai.aspx) (เว็บเจ้าของโปรเจกต์) — พอร์ต class คำนวณเดิมทีละส่วน

## แหล่งสูตร (เรียงตามความน่าเชื่อถือ)

### 1. ลูกค้า (แหล่งหลักที่ควรใช้)

| สิ่งที่ขอ | ใช้ทำอะไร |
|-----------|-----------|
| สเปกข้อความ (สุริยยาตร์, ลาหิรี, อันโตนาทีฯ, ราหู 8 กุมภ์, ทักษา) | ล็อกใน `calculationSettings.ts` — **มีแล้ว** |
| ตัวอย่าง 3–10 ชุด: วันเกิด + สถานที่ + **ผล 3 ตารางจาก myhora** | เป็น test case ผ่าน/ไม่ผ่าน |
| สูตรเป็น Excel / PDF / หนังสือ / โค้ดเก่า | แปลงเป็นฟังก์ชันใน `utils/formulas/` |
| ปฏิทิน 100 ปี (ถ้ามี) | แทนการคำนวณวันแบบสุริยยาตร์ |

**ถ้าไม่มีข้อ 2–4 → ห้ามรับปาก “ตรง myhora 100%”**

### 2. เทียบกับ myhora (วิธีปฏิบัติที่ใช้ได้)

1. กรอกข้อมูลเดียวกันใน myhora กับ NewHora  
2. บันทึกผลทีละแถว (75 แถวถ้า 3×25)  
3. ไล่สูตรทีละส่วน: ปฏิทิน → เวลาท้องถิ่น → ลาหิรี → ดาว → ราหู → ทักษา → เรือน  

นี่คือ **reverse engineering** — ใช้เวลา แต่ไม่ต้องมีสูตรเปิดเผยทั้งหมดตั้งแต่วันแรก

### 3. ชั้นคณิตศาสตร์สากล (ฐานเท่านั้น)

| ไลบรารี | ได้อะไร | ไม่ได้อะไร |
|---------|---------|------------|
| [Swiss Ephemeris](https://www.astro.com/swisseph/) / `sweph-wasm` | ตำแหน่งดาวเชิงดาราศาสตร์, ลาหิรี (ayanamsa) | สุริยยาตร์, อันโตนาที, ราหู 8 กุมภ์, ทักษาไทย |
| `astronomy-engine` | ดวงอาทิตย์/จันทร์ ฯลฯ | กฎไทยเฉพาะ |

→ ใช้คำนวณ **องศาดาว** แล้วให้ **ชั้นไทย** แปลงเป็นราศี/เรือน/ทักษา

### 4. สิ่งที่ต้องคำนวณเอง (ไม่มีใน library มาตรฐาน)

| รายการใน config | หมายเหตุ |
|-----------------|----------|
| ปฏิทินสุริยยาตร์ | แปลง ค.ศ. → วันทางโหรไทย |
| อันโตนาทีสามัญ สมผุสอาทิตย์อุทัย ปรับเวลาท้องถิ่น | ต้องมี lat/long + เวลาอาทิตย์ขึ้น |
| ราหู (๘) ราศีกุมภ์ | กฎไทย ไม่ใช่ราหู Western เสมอ |
| ทักษา ราหู = พุธกลางคืน, นับตากลาง | ทักษาไทย |
| 12 เรือน / ภพ | จากลัคนา + ระบบเรือน |

## ตอนนี้โปรเจกต์ทำอะไรอยู่

- **Output:** ตารางเดียว `ดาว | สถิตราศี` (10 แถว) — ไม่มีตารางทักษา/เรือน/25 แถว
