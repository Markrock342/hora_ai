## หลักการ (ตาม myhora)

myhora ใช้ **ปฏิทินโหราศาสตร์ไทย สุริยยาตร์** (อ้างอิงคัมภีร์สุริยยาตร์/มานัตต์) ไม่ใช่แค่ ephemeris สากล + ลาหิรีอย่างเดียว

NewHora อ่านผลแบบนี้:

1. **ชุดอ้างอิง** — `src/data/suryayat100/referenceCharts.ts` (เทียบจากราศีจักร myhora)
2. **ไฟล์ปี** — `src/data/suryayat100/years/{พ.ศ.}.json` เช่น `2549.json`
3. **ยังไม่มีวันนั้น** — fallback `ephemeris-fallback` (ประมาณ ยังไม่ตรงสุริยยาตร์)

---

## โครงไฟล์ปี

```json
{
  "beYear": 2549,
  "source": "บันทึกจาก myhora หรือปฏิทินร้อยปี",
  "days": {
    "05-21T18:31": {
      "planets": {
        "อาทิตย์": "มีน",
        "จันทร์": "กุมภ์"
      },
      "lagna": "พิจิก"
    }
  }
}
```

- คีย์วัน: `MM-DD` หรือ `MM-DDTHH:mm` (เวลาท้องถิ่น)
- ต้องครบ 10 ดาวใน `planets`

## วิธีเติมเข้า NewHora (เรียงจากเร็ว → ครบ)

### A. คุณมี myhora (แนะนำสุด)

1. Export จาก DB/โค้ดเดิม → `src/data/suryayat100/years/{พ.ศ.}.json`
2. หรือสคริปต์ดึงจาก `thai.aspx` หลังคำนวณ → บันทึก `planets` ต่อวันเวลา

### B. ไม่มี export — เทียบจากหน้าเว็บ

1. [myhora โหราศาสตร์ไทย](https://myhora.com/astrology/thai.aspx) — ตั้งค่าเดียวกับ `calculationSettings.ts`
2. กรอกวันเวลาเกิด → อ่าน **ราศีจักร ตัวเลขดำ** (ไม่ใช่ตัวแดงวันจร)
3. ใส่ใน `referenceCharts.ts` หรือ `years/2549.json`
4. ตัวอยาม: [21 พ.ค. 2549 18:31](https://myhora.com/astrology/thai.aspx) → เก็บแล้วใน `2549.json` / `referenceCharts.ts`

### C. ปฏิทินรายวัน myhora (ระวังเวลา)

[21 พ.ค. 2549](https://myhora.com/calendar/astro-suriyayas-21-05-2549.aspx) แสดงสมผุส **ณ 24:00น.** (UTC+06:42 แบบเก่า) — **ไม่เท่ากับเวลาเกิด 18:31**  
ใช้เป็นอ้างอิงปฏิทินวัน ไม่ใช่แทนดวงกำเนิด

### D. ซื้อเล่มปฏิทิน 100 ปี

สแกน / พิมพ์ลง JSON ทีละปี — ช้าแต่ได้ช่วง 2464–2563 ชัด

## หมายเหตุสำคัญ

- **ราศีจักร (thai.aspx)** = ผลที่ user ต้องการ (มีอันโตนาที + เวลาเกิด)
- **ปฏิทินรายวัน 24:00** = ตารางอีกแบบ ไม่แทนกัน
- NewHora มีชุด **21/05/2006 18:31 พระนคร** แล้ว — ทดในแอปได้เลย
