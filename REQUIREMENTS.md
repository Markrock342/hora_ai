# NewHora — Requirements (บันทึกถาวร)

## ข้อมูลนำเข้า

- ชื่อเจ้าชะตา
- วัน / เดือน / ปี เกิด (ค.ศ.)
- เวลาเกิด `HH:mm`
- สถานที่เกิด: ประเทศ, จังหวัด/รัฐ, อำเภอ/เมือง

ฟอร์ม: `BirthForm.tsx`, `InputField.tsx`, `LocationSelect.tsx`

---

## ระบบคำนวณ (คงที่ — ไม่ให้ user เลือกเพิ่ม)

| รายการ | ค่า |
|--------|-----|
| ปฏิทิน | สุริยยาตร์ |
| อายนางศา | ลาหิรี |
| เวลา | อันโตนาทีสามัญ สมผุสอาทิตย์อุทัย ปรับเวลาท้องถิ่น |
| ราหู | ๘ ราศีกุมภ์ |
| ทักษา ราหู | = พุธกลางคืน |
| ทักษา | นับตากลาง |

กำหนดใน `src/data/calculationSettings.ts`

### เทียบกับหน้าตั้งค่า myhora (ภาพอ้างอิง)

| เลือกใน myhora | NewHora |
|----------------|---------|
| ปฏิทิน สุริยยาตร์ | ✅ คงที่ |
| ลาหิรี | ✅ คงที่ |
| ลัคนา อันโตนาทีสามัญ สมผุสอาทิตย์อุทัย ปรับเวลาท้องถิ่น | ✅ คงที่ |
| ดาวเกษตร ราหู (๘) ราศีกุมภ์ | ✅ คงที่ |
| ทักษา ราหู = พุธกลางคืน | ✅ คงที่ |
| ทักษานับตากลาง | ✅ คงที่ |

**ไม่ทำ** จาก myhora: นิรายนะวิธี, ตรีวัย, มุมโยค, ชันษาจร, กาลจักร, นับวันเปลี่ยน, UI ตั้งค่าให้เลือกเพิ่ม, lat/long แสดงบนฟอร์ม (ทำทีหลังจากจังหวัด/อำเภอ)
